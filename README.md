# Day 44 — GitHub Automation Script

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A Python script that automates GitHub tasks using the
GitHub API — fetch repos, stats, create issues and more.

## How to run
```bash
pip3 install requests
python3 github_automation.py
```

## Features
- Fetch all your repos with stats
- View commit history
- Create issues via CLI
- Check repo languages breakdown
- Generate GitHub profile summary
